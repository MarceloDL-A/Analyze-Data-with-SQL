Instructions
1.
Use COUNT() and a LIKE operator to determine the number of users that have an email ending in ‘.com’.

Checkpoint 2 Passed

Hint
As an example,

SELECT * 
FROM users 
WHERE first_name LIKE '%ny';
will give you all the users whose names end in ny.

Using COUNT(*) instead of * will return the number of rows that matches the criteria.
