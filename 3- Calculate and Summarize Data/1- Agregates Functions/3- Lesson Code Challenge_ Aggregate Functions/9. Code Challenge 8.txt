When users successfully pay Codeflix (status = 'paid'), what is the average payment amount?

Checkpoint 2 Passed

Hint
Use the AVG() function after selecting the correct rows from payments:

SELECT AVG(______)
FROM payments
WHERE status = 'paid';
You don’t need GROUP BY, ORDER BY, and HAVING for this query!
